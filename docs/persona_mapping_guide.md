# 💄 K-뷰티 페르소나 ↔ 관광지 자동 매핑 기준서 (최종본)

본 문서는 한국을 방문하는 **외국인 관광객의 뷰티 소비 성향(5대 페르소나)**을 기반으로, 그들의 라이프스타일과 가장 부합하는 **오프라인 관광지(3,887곳)를 데이터 관점에서 자동 매칭한 기준과 로직**을 담고 있는 팀원 공유 및 발표용 기획 문서입니다.

## 1. 매핑 기획 개요 및 원칙
* **기획 의도**: "어떤 화장품을 사느냐(제품)"가 곧 "어떤 여행을 지향하느냐(라이프스타일)"와 직결된다는 가설을 바탕으로, 뷰티 추천 앱에서 자연스럽게 여행 코스까지 큐레이션 하는 통합 생태계를 구축합니다.
* **매핑 원칙 (다중 할당 방식)**: 특정 핫플레이스나 대형 복합쇼핑몰은 여러 페르소나의 니즈를 동시에 충족할 수 있습니다. 하나의 장소가 여러 타겟 그룹에게 맞춤형으로 큐레이션 될 수 있는 **1:N 데이터 할당 로직**을 적용했습니다.

## 2. 국가별 5대 페르소나 상세 프로필 및 매핑 기준

### 🇨🇳 1. 중국 | 효능 중심 프리미엄 케어파
* **키 메시지**: 전문가의 손길을 구현한 고기능성 안티에이징 기술력 (Home Clinic)
* **키 카피**: *"한국 클리닉의 리프팅 효과를 그대로, 집에서 완성하는 고밀도 탄력 광채."*
* **타겟 특성**: 여성 / 30~40대 / 눈에 보이는 피부 개선 효과, 프리미엄 성분, 고기능성 제품 선호
* **뷰티 포인트**: 피부과 시술 후 관리, 리프팅, 잡티 개선, 탄력 케어
* **루틴 키워드**:
  * **Brand**: 리쥬란, 메디큐브, 넘버즈인
  * **Ingredient**: PDRN, 콜라겐, 비타민C
  * **Must-buy**: 리쥬란 앰플, 메디큐브 뷰티 디바이스, 고기능 브라이트닝 세럼
* **관광지 매핑 타겟 (방한 목적)**: 백화점·쇼핑몰 중심 프리미엄 쇼핑, 전시·공연 연계 도심 문화 소비
  * **매칭 데이터 결과**: 1,142곳 큐레이션 알고리즘 연동 완료

### 🇯🇵 2. 일본 | 저자극 장벽 케어파
* **키 메시지**: 매일매일 피부가 편안한 순한 성분과 정돈된 피부결 (Daily Comfort)
* **키 카피**: *"자극 걱정 없이 맑게 차오르는 수분감, 내일이 더 기대되는 투명한 결 케어."*
* **타겟 특성**: 여성 / 20~30대 / 순한 제형, 꾸준히 쓰기 좋은 데일리 루틴, 장벽 안정감 중시
* **뷰티 포인트**: 자극 없는 진정 케어, 매일 사용 가능한 보습 제품, 피부결 정돈
* **루틴 키워드**:
  * **Brand**: 아누아, 토리든, 라운드랩
  * **Ingredient**: 시카, 히알루론산, 어성초, 세라마이드
  * **Must-buy**: 아누아 어성초 토너, 토리든 다이브인 세럼, 라운드랩 보습 크림
* **관광지 매핑 타겟 (방한 목적)**: 쇼핑몰·백화점 중심 실용형 쇼핑, 전시시설 중심 정적 문화 관광
  * **매칭 데이터 결과**: 871곳 큐레이션 알고리즘 연동 완료

### 🇹🇼 3. 대만 | 모공·쿨링 밸런스 케어파
* **키 메시지**: 습한 기후에도 끄떡없는 산뜻한 모공 진정 솔루션 (Clean & Cooling)
* **키 카피**: *"피부 온도는 낮추고 모공은 촘촘하게, 무더위 속에서도 번들거림 없는 클리어 스킨."*
* **타겟 특성**: 여성 / 30~40대 / 덥고 습한 기후에 맞는 산뜻한 제형, 모공·피지 관리, 피부 온도 진정
* **뷰티 포인트**: 클렌징 단계와 피부결 정돈 중시, 번들거림 없는 맑은 피부 표현 선호
* **루틴 키워드**:
  * **Brand**: 마녀공장, 비플레인, 브링그린
  * **Ingredient**: 녹두, PHA, 티트리, 어성초
  * **Must-buy**: 마녀공장 클렌징 오일, 비플레인 녹두 클렌징폼, 쿨링 패드
* **관광지 매핑 타겟 (방한 목적)**: 도시공원 기반 리프레시 관광, 시장 중심 로컬 라이프스타일 체험, 트렌드 소비
  * **매칭 데이터 결과**: 1,040곳 큐레이션 알고리즘 연동 완료

### 🇺🇸 4. 미국 | 즉각적 광채 추구미
* **키 메시지**: 성분으로 증명하고 결과로 보여주는 즉각적인 브라이트닝 (Instant Glow)
* **키 카피**: *"비타민과 글루타치온의 강력한 시너지, 단 한 번의 사용으로 체감하는 비현실적 광채."*
* **타겟 특성**: 남성 / 20~30대 후반 / 성분 이해도가 높고 빠른 체감 효과와 명확한 효능 선호

## 3. [어플리케이션 기능 연동 가이드] 페르소나 진단 질문 (진입 화면)
앱에 진입한 외국인 관광객이 빠르게 본인의 성향을 파악할 수 있도록 3-Step 질문을 도입합니다.

### Q1. K-뷰티 스킨케어 제품을 고를 때, 가장 중요하게 생각하는 것은? (소비 행동)
- **[A]** "강력하고 확실한 기술력!" (Home clinic 수준의 리프팅, 기미 개선) ➡️ *(중국)*
- **[B]** "내 피부는 소중하니까!" (매일 발라도 자극 없이 순한 데일리 수분) ➡️ *(일본)*
- **[C]** "끈적이는 건 딱 질색!" (가볍고 산뜻하게 모공과 열감을 잡아주는 제품) ➡️ *(대만)*
- **[D]** "어떤 성분(비타민, PDRN)이 들었지?" (과학적 근거와 명확한 결과) ➡️ *(미국)*
- **[E]** "여러 개 바를 시간 없어!" (하나로 모든 걸 케어하는 고효율 더모 솔루션) ➡️ *(홍콩)*

### Q2. 이상적인 서울 여행의 모습은 가장 가까운 것은? (관광 스타일)
- **[A]** 화려한 프리미엄 백화점에서 쇼핑하고 럭셔리한 문화 스팟 즐기기 ➡️ *(중국, 홍콩)*
- **[B]** 무더위를 피해 트렌디한 팝업 스토어나 로컬 시장 둘러보기 ➡️ *(대만)*
- **[C]** 복합문화공간을 걷고 K-컬처 액티비티를 강렬하게 체험하기 ➡️ *(미국)*
- **[D]** 조용한 전시관이나 한적한 실내에서 번잡함 없이 차분하게 머물기 ➡️ *(일본)*

### Q3. K-뷰티 쇼핑으로 완성하고 싶은 나의 완벽한 피부 상태는? (최종 기대 효능)
- **[A]** 전문 관리샵을 다녀온 듯 탄력 넘치고 결점 없는 **[고밀도 광채 윤광 피부]** ➡️ *(중국)*
- **[B]** 인위적이지 않게 수분을 꽉 머금어 차분하고 맑은 **[편안한 투명 물광 피부]** ➡️ *(일본)*
- **[C]** 온도를 낮추고 번들거림을 잡아 산뜻하고 매끄러운 **[모공 리스 보송 피부]** ➡️ *(대만)*
- **[D]** 체계적인 성분 케어로 티 없이 밝게 톤업된 **[즉각적인 브라이트닝 광채 피부]** ➡️ *(미국)*
- **[E]** 바쁜 일정 속에서도 방금 팩을 뗀 것처럼 **[단기간에 생기 도는 효율적 피부]** ➡️ *(홍콩)*
